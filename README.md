# Go Server Monitor 

Простой мониторинг состояния системы на Go, который проверяет:

- Загрузку CPU  
- Использование памяти  
- Использование диска  

 При достижении критических значений выводит предупреждения в терминал.

---

##  Функционал

- Отслеживание состояния CPU, RAM и диска
- Конфигурируемые пороговые значения
- Простое уведомление в терминал
- Мониторинг в реальном времени каждые 5 секунд
